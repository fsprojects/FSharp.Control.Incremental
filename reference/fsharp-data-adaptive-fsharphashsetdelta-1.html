HashSetDelta<'T> Type

Namespace: FSharp.Data.Adaptive
Assembly: FSharp.Data.Adaptive.dll
Base Type: ValueType
All Interfaces: IEquatable<HashSetDelta<'T>> , IEnumerable<SetOperation<'T>> , IEnumerable
Kind: Struct

Represents the difference of two HashSets. Internally uses reference counts to represent deltas and provides convenient combine functions.
